# SIGNAL FORGE

AI-powered trading signal intelligence platform powered by MiMo V2.5 + Hermes Agent.

## Architecture
- **Sentinel** — Scans 200+ market feeds in real-time
- **Arbitrage** — Detects cross-exchange price inefficiencies within milliseconds
- **Risk Guard** — Dynamic position limits and drawdown controls
- **Executor** — Routes orders through optimal liquidity venues

## Tech Stack
- MiMo V2.5 (chain-of-thought signal reasoning)
- Hermes Agent (orchestration layer)
- Claude series (market analysis)
